# Blueprinter Website

Marketing one-pager for [Blueprinter](https://blueprinter.io) — an AI-first web data extraction platform for e-commerce intelligence.

## Tech Stack

- Next.js 16 (App Router, TypeScript)
- Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

## Deployment

Static export to Cloudflare Pages:

```bash
npm run build   # outputs to out/
```
